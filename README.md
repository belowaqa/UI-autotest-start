# UI-autotest-start
## 🐳 Запуск тестов в Docker

Для изоляции окружения и исключения проблем с зависимостями браузеров, тесты можно запускать внутри Docker-контейнера.

### 1. Сборка Docker-образа
Перед первым запуском (или после изменения `requirements.txt`) необходимо собрать образ. Находясь в корне проекта, выполни:

```bash
docker build -t playwright-tests .

### 2. Запуск теста
docker run --rm playwright-tests

Этот образ весит 6.22 гб потому что базовый образ microsoft весит около 4 гб и в него зашиты все три браузера
